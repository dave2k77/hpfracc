# Fractional ODE Examples

This directory contains minimal operator and fixed-step Caputo FDE examples.

Run the operator example:

```bash
python examples/fractional_ode/caputo_operator.py
```

Run the solver example:

```bash
python examples/fractional_ode/caputo_solver.py
```

Run the synthetic training example:

```bash
python examples/fractional_ode/train_neural_fode.py
```

Run the scalar-grid probabilistic calibration example:

```bash
python examples/fractional_ode/probabilistic_calibration.py
```
