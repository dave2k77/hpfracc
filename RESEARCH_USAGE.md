# Research Usage Policy

HPFRACC is a research software framework for fractional-order scientific
computing, differentiable modelling, and future synthetic brain-dynamics
simulation.

It is intended for methodological research, simulation studies, numerical
experiments, and exploratory scientific analysis. It is not a clinical device,
is not validated for diagnosis or treatment, and should not be used as a
substitute for empirical biomedical assessment or externally validated
subject-specific modelling.

Published or reported results based on HPFRACC should record:

- HPFRACC version.
- Commit hash or release tag, if available.
- Backend and hardware context.
- Operator, solver, and model configuration.
- Random seed or key lineage where stochasticity is involved.
- Experiment manifest or equivalent reproducibility record.
- Validation status of operators and solvers used.
